# UndertaleForWii
A Remake of Undertale For The Wii. Made in Unity 3.5.6 For Wii
It can be ported for other consoles like the 3DS although you will experience a lot of bugs 

List of Features
- Save Point
- Flowey Encounter (Not complete yet)
- WIP Battle System
- Full Intro (With no text)
- Sound Test
- Toriel Encounter (In-Battle)
- Basic Movement
- Music
- Sound Effects
- First 3 Rooms


# Bugs

- Missing Interactions

- Some UI bugs
